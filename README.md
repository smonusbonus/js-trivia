# trivia.js
trivia.js is a fun way to test your Javascript knowledge and become a better programmer.

## Hacking
``` bash
# install dependencies
yarn install

# start the node/express server
yarn run start

# start the webpack dev serve with hot reload
yarn run dev

# run unit tests
yarn run unit

# run all tests
yarn test
```
